Introduction
引言

C语言是一种较早的程序设计语言，诞生于1972年的贝尔实验室。1972 年，Dennis Ritchie 设计了C语言，它继承了B语言的许多思想，并加入了数据类型的概念及其他特性。 尽管C 语言是与 UNIX 操作系统一起被开发出来的，但它不只支持UNIX。C是一种通用（广泛可用）的编程语言。

C语言是一门面向过程的计算机编程语言，与C++、C#、Java等面向对象编程语言有所不同。C语言的设计目标是提供一种能以简易的方式编译、处理低级存储器、仅产生少量的机器码以及不需要任何运行环境支持便能运行的编程语言。C语言描述问题比汇编语言迅速、工作量小、可读性好、易于调试、修改和移植，而代码质量与汇编语言相当。C语言一般只比汇编语言代码生成的目标程序效率低10%-20%。因此，C语言可以编写系统软件.

c_and_cpp
C语言和C++

​C语言与C++的主要区别在于编程范式与功能扩展：C语言是面向过程的编程语言，强调函数调用和步骤分解，适合底层开发（如操作系统、嵌入式系统）；而C++在C的基础上扩展为多范式语言，支持面向对象（类、继承、多态）、泛型编程（模板）等特性，更适合构建复杂系统（如游戏引擎、图形界面）

C++是C语言的超集，在语法层面高度兼容C语言，保留了C的高效性与底层控制能力，同时通过面向对象（类、继承、多态）、泛型编程（模板）等特性扩展了编程范式

variables
变量

C语言中的变量是程序中存储数据的基本单元，通过声明与类型定义为特定数据类型（如整型、int、浮点型float、字符型char等）分配内存空间。变量需在使用前声明，可初始化赋值（如int age = 25;），其作用域分为局部变量​（限于函数或代码块内部）和全局变量​（程序全局可见），生命周期则取决于存储类别（如自动变量随函数调用结束销毁，静态变量static持久存在）。

变量的数据类型决定存储内容及内存大小（如int占4字节，char占1字节），合理选择类型可提升效率；命名规则需以字母/下划线开头，区分大小写，并避免关键字。使用时需注意作用域冲突​（如局部变量覆盖全局变量）和初始化必要性​（未初始化变量可能含随机值，导致程序异常）

data_types
数据类型

C语言中的数据类型主要分为基本类型、派生类型和用户自定义类型

    一、基本数据类型
  • 整型 int
    int i = 0;
  • 字符型 char
    char c = 'char';
  • 浮点型 float、double
    float f = 1.0;
    double d = 1.2;
  • 空类型 void
    二、派生数据类型
  • 指针 *
    int* n = NULL;
  • 数组 []
    a[2] = {0, 1, 2}
  • 结构体 struct
    struct Point { int x; int y; };
  • 联合体 union
    union Number { int i; float f; };
    三、用户自定义类型
  • 枚举 enum
    enum Weekday {Mon, Tue, Wed};
  • 类型别名 typedef
    typedef struct Point Coordinate;
